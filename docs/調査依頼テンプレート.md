## 13. 修正指示の出し方ルール
AI は次の形式で返すこと。

- ファイル名を明示する
- `○行目〜○行目を置き換えてください`
- `○○関数を丸ごと置き換えてください`
- `○行目の下に追加してください`

あいまいな言い方は避けること。

---

## 14. AI への補足
- ユーザーは Python 初心者
- コードは基本コピペで反映する
- 解説は原則不要
- NORMAL ログだけで原因が分からない場合は DEBUG に切り替える
- `main` では直接作業しない
- 普段の作業継続元ブランチは `move-to-users-silve`
